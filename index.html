Найти количество положительных, отрицательных и равных нулю случайных элементов массива. 

                    
                            from random import random
                            neg = pos = zero = 0
                            a = []
                            for i in range(20):
                                n = int(random() * 10) - 5
                                a.append(n)
                                print(n, end=' ')
                                if n > 0:
                                   pos += 1
                             elif n < 0:
                                    neg += 1
                                else:
                                    zero += 1
                                print("\nПоложительных: ", pos)
                                print("Отрицательных: ", neg)
                                print("Равных нулю: ", zero)
                                Результат:
                                -5 2 3 -1 4 1 1 4 -3 -3 -2

Элементы матрицы NxM вычисляются по формуле A[i,j] = sin(N*i+M*j) при индексации с единицы или по формуле A[i,j] = sin(N*(i+1)+M*(j+1)) при индексации с нуля. Если полученный таким образом элемент матрицы отрицателен, то заменить его на 0. Вывести конечную матрицу на экран. 

                    
                            from math import sin
                            N = 3
                            M = 4
                            mtx = []
                            for i in range(N):
                                a = []
                                for j in range(M):
                                    a.append(sin(N*(i+1) + M*(j+1)))
                                mtx.append(a)
                             for i in range(N):
                             for j in range(M):
                                    if mtx[i][j] < 0:
                                        mtx[i][j] = 0
                                     print("%7.0f" % mtx[i][j], end='')
                                 else:
                                     print("%7.2f" % mtx[i][j], end='')
                             print()

Вывести на экран индексы всех минимальных элементов матрицы.

                    
                            from random import random
                            M = 7
                            N = 5
                            a = []
                            for i in range(N):
                             b = []
                             for j in range(M):
                                    b.append(int(random()*50) - 25)
                                    print("%4d" % b[j], end='')
                                a.append(b)
                                print()
                            min_mx = 25
                            for i in range(N):
                             min_i = min(a[i])
                             if min_i < min_mx:
                                    min_mx = min_i
                            print(min_mx)
                            for i in range(N):
                                for j in range(M):
                                if min_mx == a[i][j]:
                                print('Row: %d, col: %d' % (i+1,j+1))
